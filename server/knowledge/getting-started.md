# Getting started

## Sign up

1. Go to the app homepage and click Get started.
2. Confirm your email within 24 hours.
3. Complete the onboarding wizard (workspace name, timezone).

## First project

After sign-up, create a project from the dashboard. Each project has its own API keys and usage limits.

## API keys

Generate keys under Settings → API. Never commit keys to git. Rotate keys immediately if exposed.

## Local development

Use our REST API at `https://api.example.com/v1`. Rate limit on free tier: 60 requests/minute.

## Support hours

Chat support is available 24/7 for Pro and Team. Starter plans get email support with 1-business-day response.
